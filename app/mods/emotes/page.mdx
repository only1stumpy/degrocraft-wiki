# Эмоции

Ваш персонаж может танцевать, махать рукой и проявлять чувства благодаря моду **Emotecraft**.

### Установка

1. Установите мод **Emotecraft** для Fabric или Forge.
2. Скачайте **дополнительный пак эмоций (1000+ анимаций)** [https://spemotes.com/download/].
3. Распакуйте содержимое пака в папку `emotes` в корне вашей папки с игрой (`.minecraft/emotes`). Если папки нет - создайте ее.

### Как использовать

- Нажмите клавишу **B** (по умолчанию), чтобы открыть колесо выбора эмоций.
- В настройках мода вы можете назначить горячие клавиши для самых любимых танцев.
