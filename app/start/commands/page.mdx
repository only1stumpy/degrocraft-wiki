# **Команды**

Ниже приведен список основных команд, доступных каждому игроку на сервере **DegroCraft**.

| Команда                          | Описание                                                                                                     |
| :------------------------------- | :----------------------------------------------------------------------------------------------------------- |
| `/msg [ник игрока]`              | Написать личное сообщение игроку                                                                             |
| `/skin set [ник]`                | Поставить себе скин по нику                                                                                  |
| `/skin url [ссылка]`             | Поставить себе скин по ссылке на файл со скином.                                                             |
| `/disc burn [ссылка] [название]` | Создать пластинку с вашей музыкой, держа ее в руке.                                                          |
| `/disc erase`                    | Удалить музыку с вашей пластинки                                                                             |
| `/itf togglemode visibility`     | Включить/выключить режим изменения рамок, при котором клик правой кнопки мыши делает рамку видимой/невидимой |
