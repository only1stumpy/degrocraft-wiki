# Взаимодействие с миром

Добавляет возможность принимать реалистичные позы для отдыха или общения.

### Основные возможности

- **Сесть на блок:** Нажмите **ПКМ** (правую кнопку мыши) по любой ступеньке, полублоку или ковру пустой рукой.
- **Сесть на игрока:** Нажмите **ПКМ** по голове другого игрока, чтобы залезть к нему на плечи.
- **Позы через команды:** Вы можете использовать команды для отдыха в любом месте.

### Команды

| Команда      | Описание                       |
| :----------- | :----------------------------- |
| `/sit`       | Сесть на землю                 |
| `/lay`       | Лечь на землю                  |
| `/bellyflop` | Лечь на живот                  |
| `/spin`      | Начать вращаться (как в мемах) |
| `/crawl`     | Включить режим ползания        |

---

Чтобы слезть с игрока или встать, нажмите клавишу **Shift**.
