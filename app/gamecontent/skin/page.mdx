# Смена скина

На нашем сервере вы можете установить себе любой скин, даже если вы играете с пиратской версии игры.

### Основные команды

| Команда        | Описание                                              |
| :------------- | :---------------------------------------------------- |
| `/skin [ник]`  | Установить скин другого игрока по его нику            |
| `/skin [url]`  | Установить скин по прямой ссылке на .png файл         |
| `/skin clear`  | Сбросить скин к стандартному                          |
| `/skin update` | Обновить текущий скин (если владелец ника его сменил) |

<Callout type="info" emoji="✨">
  После ввода команды скин обновится мгновенно. Если этого не произошло,
  попробуйте перезайти на сервер.
</Callout>

### Где искать скины?

Вы можете найти скины на сайтах вроде [Namemc](https://namemc.com) или [MinecraftSkins](https://www.minecraftskins.com). Просто скопируйте ник игрока, чей скин вам понравился, и введите `/skin ник`.
