# Команды

Ниже приведен список основных команд, доступных каждому игроку на сервере **DegroCraft**.

| Команда                      | Описание                                                           |
| :--------------------------- | :----------------------------------------------------------------- |
| `/msg [ник]`                 | Написать личное сообщение игроку                                   |
| `/r [текст]`                 | Быстро ответить на последнее личное сообщение                      |
| `/skin set [ник]`            | Поставить себе скин по нику                                        |
| `/skin url [ссылка]`         | Поставить себе скин по прямой ссылке на файл                       |
| `/skin clear`                | Сбросить установленный скин                                        |
| `/disc burn [url] [назв.]`   | Записать музыку на пластинку в руке — [Гайд](/mods/voice)          |
| `/disc erase`                | Стереть запись с пластинки — [Гайд](/mods/voice)                   |
| `/itf togglemode visibility` | Переключатель видимости рамок — [Гайд](/gamecontent/itf)           |
| `/nophantoms toggle`         | Включить или выключить фантомов — [Гайд](/gamecontent/phantoms)    |
| `/mention toggle`            | Включить или отключить звуки тегов — [Гайд](/gamecontent/mentions) |
| `/sit`                       | Сесть на землю — [Гайд](/gamecontent/gsit)                         |
| `/lay`                       | Лечь на землю — [Гайд](/gamecontent/gsit)                          |
| `/bellyflop`                 | Лечь на живот — [Гайд](/gamecontent/gsit)                          |
| `/emotecraft`                | Настройка меню эмоций — [Гайд](/mods/emotes)                       |

---

Все команды поддерживают автозаполнение через клавишу **Tab**.
