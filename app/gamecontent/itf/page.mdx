# Невидимые рамки

Механика для эстетов и декораторов, позволяющая делать рамки прозрачными.

- **Команда:** `/itf toggle visibility`
- **Как использовать:** Команда позволяет включить/выключить режим изменения рамок, при котором клик правой кнопки мыши делает рамку видимой/невидимой. Рамка станет невидимой, оставляя висеть только предмет.
- **Ограничение:** Игрокам разрешено только переключение видимости (toggle visibility). Остальные функции плагина ограничены.
