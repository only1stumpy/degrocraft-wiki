# Взаимодействие с миром

Добавляет возможность принимать реалистичные позы для отдыха или общения.

**Доступные возможности для игроков:**

- **Сесть на блок:** Нажмите **ПКМ**
  (правую кнопку мыши) по любой ступеньке, полублоку или ковру пустой рукой.
- **Сесть на игрока:** Нажмите **ПКМ** по голове другого игрока, чтобы залезть к
  нему на плечи.
